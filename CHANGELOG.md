# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-15

### Added

- `useDomTransfer` Hook — declarative API with `sourceRef` / `targetRef`
- `transferDom` / `restoreDom` — imperative API
- FLIP animation with configurable CSS transition
- Fiber `stateNode` swap (preserves React tree structure)
- Placeholder element to prevent source layout collapse
- Fallback timeout for `transitionend`
- Dev-mode warnings for missing refs and empty containers
